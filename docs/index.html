fb_mhash 1.0

Richard Kotal, info@waplikace.cz, www.aolserver.cz

The UDF module implements interface to mhash library (http://mhash.sourceforge.net).
The module was tested with 0.9.4 version of mhash library.

fb_mhash has three categories of the commands:
  • key generator UDF - generate key from password (mcrypt,...)
  • hash UDF - create hash string
  • hmac UDF - create hmac string

Small samples: 
select first 1 mhash_keygen('hex','MCRYPT',mycol,'MD5','',10,'',0),mycol from mytable;
expected result: "Hello world", "3e25960a79dbc69b674c" 

select first 1 mhash_hash('hex','MD5',mycol),mycol from mytable;
expected result: "Hello world", "3e25960a79dbc69b674cd4ec67a72c62" 

select first 1 mhash_hmac('base64','MD5',mycol,'password'),mycol from mytable;
expected result: "Hello world", "MyPQK3X37QIxu6NfSOK68g=="
