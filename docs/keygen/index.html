Keygen

UDF key generate

command:mhash_keygen('outfmt','algo_name','password','hash_algo_1','hash_algo_2',size_of_output_key,'salt_string',count)
parameters: outfmt - a type of the output format for the generated key :: hex: hexadecimal string; raw: raw (binary) string; base64: binary base64 encoded string
algo_name - the name of the keygen algorithm: HEX, ASIS, S2K_SALTED, ...
password - an input string
hash_algo_1 - first hash algorithm used by the keygen algorithm: MD5,SHA1,...
hash_algo_2 - second hash algorithm used by the keygen algorithm or the empty string (see mhash manual)
size_of_output_key - the size of the output generated key
salt_string - a salt used by the keygen algorithm
count - count how many the keygen algorithm hashs repeatedly the password
result:the key generated by the specified keygen algorithm from the password. Otherwise, empty string is returned on failure.
example: select first 1 mhash_keygen('base64','MCRYPT',mycol,'MD5','',10,'',0),mycol from mytable; = "Hello world", "PiWWCnnbxptnTA=="
select first 1 mhash_keygen('hex','MCRYPT',mycol,'MD5','',10,'',0),mycol from mytable; = "Hello world", "3e25960a79dbc69b674c"
select first 1 mhash_keygen('hex','S2K_SALTED',mycol,'SHA1','',20,'salt',0),mycol from mytable; = "Hello world", "a1be1548560896f5a655c3577769d7cfced5925b"
